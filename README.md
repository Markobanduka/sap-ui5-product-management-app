
# Product Management

SAP UI5 / Fiori-style Product Management application built using the MVC architecture and consuming an OData service.

The application allows users to view, create, edit, delete, and manage products, while demonstrating key SAP UI5 enterprise concepts such as routing, fragments, data binding, sorting, filtering, and grouping.


## Tech Stack

• SAP UI5 / Fiori

• JavaScript (ES6)

• OData V2

• XML Views & Fragments

• JSONModel

• UI5 Routing
